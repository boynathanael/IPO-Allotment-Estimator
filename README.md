# IPO-Allotment-Estimator
An automated Excel VBA-based tool designed to help retail investors and traders estimate their potential share allotment in an IPO, fully compliant with the latest Indonesian regulation **SEOJK No. 25/SEOJK.04/2025**.

## Key Features
- **Automated Tier Classification:** Instantly categorizes IPOs into 5 tiers based on market capitalization.
- **Dynamic Adjustment Logic:** Calculates adjusted share allocations for oversubscription scenarios.
- **Smart Highlighting:** Visual cues to identify the most likely allotment range based on user participation assumptions.
- **User-Friendly Interface:** Built with a custom VBA UserForm for seamless data entry.

## How to Use
1. Download the `.xlsm` file from this repository.
2. Open the file in Microsoft Excel and **Enable Macros**.
3. Launch the estimator using the trigger button on the main sheet.
4. Input the IPO parameters and predicted investor count.
5. Click **"Generate Table"** to view your results.

## Disclaimer
This tool provides estimations based on public regulations. Final allotments are determined solely by the Lead Underwriter and the Electronic IPO system.
